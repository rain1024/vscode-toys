## General Shortcuts

| Action        | Mac           | Windows/Linux |
|---------------|---------------|---------------|
| Open File     | `Cmd + O`     | `Ctrl + O`    |
| Save File     | `Cmd + S`     | `Ctrl + S`    |
| Find          | `Cmd + F`     | `Ctrl + F`    |
| Replace       | `Cmd + H`     | `Ctrl + H`    |
| Undo          | `Cmd + Z`     | `Ctrl + Z`    |
| Redo          | `Cmd + Y`     | `Ctrl + Y`    |

## Navigation Shortcuts

| Action              | Mac                       | Windows/Linux             |
|---------------------|---------------------------|---------------------------|
| Go to Line          | `Cmd + G`                 | `Ctrl + G`                |
| Close Tab           | `Cmd + W`                 | `Ctrl + W`                |
| Open Next Editor    | `Cmd + Option + Right`    | `Ctrl + K, Ctrl + Right`  |
| Open Previous Editor| `Cmd + Option + Left`     | `Ctrl + K, Ctrl + Left`   |

## Code Editing Shortcuts

| Action            | Mac                   | Windows/Linux        |
|-------------------|-----------------------|----------------------|
| Comment Line      | `Cmd + /`             | `Ctrl + /`           |
| Indent Line       | `Tab`                 | `Tab`                |
| Unindent Line     | `Shift + Tab`         | `Shift + Tab`        |
| Duplicate Line    | `Cmd + D`             | `Ctrl + D`           |
| Add Cursor Below  | `Cmd + Option + Down` | `Ctrl + Alt + Down`  |
| Add Cursor Above  | `Cmd + Option + Up`   | `Ctrl + Alt + Up`    |
| Delete Line       | `Cmd + Shift + K`     | `Ctrl + Shift + K`   |

## View Shortcuts

| Action            | Mac           | Windows/Linux |
|-------------------|---------------|---------------|
| Toggle Sidebar    | `Cmd + B`     | `Ctrl + B`    |
| Toggle Terminal   | `Cmd + J`     | `Ctrl + ~`    |
| Zoom In           | `Cmd + +`     | `Ctrl + +`    |
| Zoom Out          | `Cmd + -`     | `Ctrl + -`    |
| Split Editor      | `Cmd + \`     | `Ctrl + \`    |
| Toggle Full Screen| `Ctrl + Cmd + F` | `F11`      |
| View Zen Mode       | `Cmd + K, Z`              | `Ctrl + K, Ctrl + Z`      |

## Debugging Shortcuts

| Action            | Mac           | Windows/Linux |
|-------------------|---------------|---------------|
| Start/Continue Debugging | `F5`  | `F5`          |
| Step Over         | `F10`         | `F10`         |
| Step Into         | `F11`         | `F11`         |
| Step Out          | `Shift + F11` | `Shift + F11` |
| Stop Debugging    | `Shift + F5`  | `Shift + F5`  |

## Customizing Shortcuts

You can customize these shortcuts or add new ones by going to `File > Preferences > Keyboard Shortcuts` or pressing `Cmd + K, Cmd + S` on Mac or `Ctrl + K, Ctrl + S` on Windows/Linux.
